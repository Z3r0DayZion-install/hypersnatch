# HyperSnatch UI Visual System

## Design goals
The UI should feel:
- forensic
- elite
- technical
- dark
- readable for long sessions
- fast to scan under pressure

## Overall aesthetic
Use a dark command-center aesthetic:
- background: near-black / charcoal
- panels: layered graphite / soft glass
- accent: electric cyan + signal green + muted amber
- danger / blocked states: sharp red
- typography: clean mono + clean sans mix

## Recommended palette
- Background: #0A0D12
- Panel: #121821
- Panel Alt: #171F2B
- Border: #263242
- Primary Accent: #20C7FF
- Success: #23D18B
- Warning: #F5B83D
- Danger: #FF5C7A
- Text Primary: #EAF2FF
- Text Secondary: #9FB0C7
- Muted: #607089

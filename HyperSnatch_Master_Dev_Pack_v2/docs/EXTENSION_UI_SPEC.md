# Browser Extension UI Spec

The extension should be a compact capture console.

## Popup layout
- hostname
- player detected
- quick status badge
- streams detected count
- protocol detected
- latest candidate URL
- toggles:
  - network
  - JS trace
  - MSE
  - DOM snapshot
- actions:
  - send to desktop
  - export session
  - pin site rule
  - open detailed analysis

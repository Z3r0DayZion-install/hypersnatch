## Reproduce It Yourself

```powershell
# Local (PowerShell)
.\scripts\reproduce.ps1

# Docker (any platform)
docker build -f Dockerfile.repro -t hypersnatch-repro .
docker run --rm hypersnatch-repro
```

See [REPRODUCIBILITY.md](REPRODUCIBILITY.md) for details.

## Verify a Release

```powershell
# PowerShell
.\verify.ps1 -FilePath .\HyperSnatch-Setup-1.2.1.exe

# Node.js (cross-platform)
node verify_node.js HyperSnatch-Setup-1.2.1.exe
```

See [VERIFY_RELEASE.md](VERIFY_RELEASE.md) for the full walkthrough.


# Autonomous Pattern Learning Engine

Goal:
Allow HyperSnatch to learn new sites automatically.

Process:

1. Successful decode recorded
2. Pattern extracted
3. Rule generalized
4. Stored in fingerprint database

Example Rule:

site: example.com
player: videojs
pattern: sources[0].src

Next Visit:
Stream decoded instantly.

Long Term Result:
Massive database of site extraction patterns.

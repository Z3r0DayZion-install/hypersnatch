# Module Ownership Map

Core modules:
- graph: entity merge, neighborhood expansion, lineage
- attribution: provider and hypothesis scoring
- fingerprint: operator similarity modeling
- replay: chronological reconstruction and deltas
- counterfactual: assumption toggles and delta analysis
- challenge: adversarial counter-arguments
- command: orchestration and dispatch
- provenance: source tracing
- xai: explanation trees and confidence contributions
- heatmap: cluster density and risk concentration
- memory: analyst annotations and cross-case state
- narrative: spread dynamics and role classification
- behavior: operator habits and deployment windows
- forecast: future event probability hypotheses
- copilot: guided query suggestions
